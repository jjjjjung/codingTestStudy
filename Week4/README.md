*n*을 입력받고 1부터 *n*까지 반복문을 돌며 해당 수의 자리수를 리스트로 만들어준다

```py
num = list(map(int, str(i)))
```
그러고 해당 수와 자리수의 합을 더해서 *n*과 같으면 반복문을 종료하고 출력해준다.

처음에 *break*를 거는걸 까먹어서 틀렸는데 가장 작은 생성자를 구해야 하므로 원하는 답이 나오면 잊지말고 *break* 걸어줄 것

```py
if ans == n:
        res = i
        break
```
